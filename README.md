# EXCEPTION:0xDEAD??
A full-horror Minecraft mod where the system fails, the world corrupts, and something inside the code starts watching you.

## Overview
EXCEPTION:0xDEAD?? is a digital-horror Minecraft mod focused on system corruption, psychological tension, and world instability. This is not a “creepy ambience” mod — it is designed to make the player question whether Minecraft is still functioning correctly.

## Features
- System-level horror events
- Fake errors, corrupted UI, and unstable menus
- Glitched entities and unpredictable world behavior
- Environmental distortion and visual corruption
- Narrative elements hidden in logs, files, and world data
- Multiple endings or outcomes based on player actions

## Installation
1. Install Fabric or Forge (depending on your version).
2. Download the latest release from the Releases tab.
3. Place the `.jar` file into your `mods` folder.
4. Launch Minecraft.
5. If the game behaves strangely… that’s intentional.

## Warnings
This mod contains:
- Fake system messages
- Visual glitches
- Audio distortion
- Psychological horror
- Fourth-wall breaks

It does NOT modify real files or harm your system.
